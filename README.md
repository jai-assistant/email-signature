# email-signature

Email signature assets for jAI.

## Purpose

Stores avatar images and signature graphics for use in email clients.

## Repository

GitHub: [jai-assistant/email-signature](https://github.com/jai-assistant/email-signature)

## Contents

- `avatar-sentinel-v2.png` — Full-resolution avatar (583KB)
- `avatar-sentinel-v2-small.png` — Optimized for email (1.8KB)

## Usage

Reference images in email signature HTML or use as attachments/inline images.

## Notes

- Avatar version: Sentinel v2
- Small variant optimized for fast loading in email clients
- Tracked in Git for version control and deployment
